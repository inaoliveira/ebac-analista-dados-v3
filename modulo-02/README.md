# Módulo 2: Configuração de Ambiente e Linux

Este módulo abordou a preparação do ambiente de desenvolvimento e comandos essenciais de terminal para um Analista de Dados.

## Ferramentas Utilizadas:
- **WSL2 (Ubuntu):** Ambiente Linux rodando sobre o Windows 11.
- **VS Code:** Editor principal com integração remota via extensão WSL.

## Principais Comandos Praticados:
- Navegação no sistema de arquivos (`cd`, `ls`, `pwd`).
- Manipulação de arquivos e pastas (`mkdir`, `touch`, `cp`, `mv`).
- Permissões e visualização (`chmod`, `cat`, `head`, `tail`).## Atividade Prática: Scripting e Permissões
Desenvolvimento de um script .sh com gestão de permissões via chmod.
